
C++ graph class containing the implementation of the following algorithms:
- BFS
- DFS
- Biconnected Components
- Strongly Connected Components
- Topological Sort
- Havel Hakimi
- Bridges in a Graph
- Minimum Spanning Tree (Kruskal)
- Dijkstra
- Bellman Ford
- Edmonds Karp
- Graph diameter
- Roy Floyd 


Bonus: disjoint set forest class, used in Kruskal
